Stellar block is a module where you can create an block that includes a parallax image. This module also integrates with <a href="http://drupal.org/project/imagecache">Imagecache</a> for dynamic image sizing and manipulation.

How to use:
1. Enable Stellar block module.</li>
2. Navigate to admin/build/block.</li>
3. Click the "Add Stellar block" link in the local tasks menu.</li>
4. Configure your Stellar block, and save.
